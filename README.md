# AI Ticket Categorization System

## Overview
An AI-based system that automatically categorizes IT support tickets using NLP and Machine Learning.

## Features
- Automatic ticket classification
- Text preprocessing using NLP
- Model training and evaluation
- Accuracy comparison

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Pandas
- NumPy
- Matplotlib

## Project Structure
- data/
- models/
- static/
- templates/
- create_admin.py
- nexus.db
- requirements.txt
- train_models.py
- app.py

## How to Run
1. Clone the repository
2. Install requirements
3. Run the main file
